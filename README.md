# FoxDockerScript
FoxDockerScript - Hosting Control Panel sử dụng Docker và Traefik, được xây dựng bằng Go và Vue.js.

## Cấu trúc dự án
- `backend/`: Go Backend (REST API, Docker SDK, Proxy Logic)
- `frontend/`: Web Admin UI
- `deploy/`: Scripts cài đặt và cấu hình
- `storage/`: Dữ liệu hệ thống và dự án
